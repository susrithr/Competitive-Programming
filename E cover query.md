#E - Cover query 
Time Limit: 2 sec / Memory Limit: 1024 MiB

Score : 
450 points

Problem Statement
There are N cells arranged in a row from left to right.
The i-th cell from the left (1≤i≤N) is called cell i.
Initially, all cells are painted white.
Process 
Q queries in order.
The i-th query (1≤i≤Q) is as follows:
Two integers (L i,R i) (1≤L i≤R i≤N) are given.
Paint all cells L i,L i+1,…,R i black.
Here, cells that were painted white are now painted black, and cells that were painted black remain as is.
After that, find the number of cells (among the 
N cells) that are painted white.
